
# Isometric Village Game

Um jogo de construção de vila em perspectiva isométrica com NPCs.

## Estrutura do Projeto

```
├── assets/           # Recursos gráficos
├── src/             # Código fonte
│   ├── config/      # Configurações
│   ├── core/        # Core do jogo
│   ├── entities/    # Classes de entidades
│   ├── renderer/    # Sistema de renderização
│   ├── scenes/      # Cenas do jogo
│   └── ui/          # Interface do usuário
```

## Componentes Principais

- **Game**: Gerenciador principal do jogo
- **Grid**: Sistema de grade isométrica
- **NPCs**: Sistema de personagens não jogáveis
- **BuildPanel**: Interface de construção
